Debian Install Packages

64-bit: gvpngate_0.54-1_amd64.deb
32-bit: gvpngate_0.54-1_i386.deb

64-bit Debian package has been tested on:

- Debian 7.9 (Wheezy)               - Debian 8.4 (Jessie)
- Ubuntu 14.04 (Trusty Tahr)        - Ubuntu 15.10 (Wily Werewolf)
- PinguyOS 14.04                    - Linux Mint 17.3 Cinnamon (Rosa)

32-bit Debian package has been tested on:

- Debian 7.10 (Wheezy)

Ubuntu requires the "universe" repository and the program icon dosen't show up in Dash until after logging out of Ubuntu and back in again.


RPM Install Packages

64-bit: gvpngate-0.54-1.x86_64.rpm
32-bit: gvpngate-0.54-1.i586.rpm

64-bit RPM package has been tested on:

- Fedora 23 Workstation             - openSUSE 13.2 (Harlequin)
- CentOS 7.2-1511

32-bit RPM package has been tested on:

- Fedora 23 Workstation

CentOS requires the "EPEL" repository.
